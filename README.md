# Dungeon Mapper MM1

A retro-style dungeon mapping editor for classic dungeon crawler adventures. The app is built with Vite + React + TypeScript and focuses on:

- 16x16 tile editing for dungeon and wilderness maps
- terrain, feature, and encounter markers per tile
- side-state editing for walls, doors, gates, windows, secret passages, and open edges
- link tracking for map-to-map connections
- local persistence with JSON import/export for sharing between computers

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
